 The Number Object

Number Constructor
Number(arg) Converts arg to a number.
new Number(arg) Creates a new number object using the number-converted arg as a value.
MAX_VALUE Returns the largest number, 1.7976931348623157e308
MIN_VALUE Returns the smallest number, 4.9e-324.
NaN Returns the not-a-number constant.
NEGATIVE_INFINITY Returns negative infinity, -1/0.
POSITIVE_INFINITY Returns positive infinity, 1/0.
